# Motte.ai Documentation

Motte.ai is a powerful web platform for training language models using reinforcement learning from human feedback (RLHF). Train models, monitor metrics in real-time, and optimize costs efficiently.

## Overview

The Motte.ai platform provides:

- **Web-based interface** - No installation required, works in any browser
- **Real-time training metrics** - Live charts powered by Weights & Biases
- **Cost transparency** - Clear $80/hour pricing with real-time tracking
- **Model fine-tuning** - Support for popular models (Llama, Qwen, Gemma)
- **Advanced monitoring** - Terminal streaming, progress tracking, and alerts

## Quick Start

1. **Sign up** at [Motte.ai](https://motte.ai) and verify your email
2. **Add billing information** to enable $80/hour GPU training
3. **Navigate to [Create Run](https://motte.ai/runs/create)**
4. **Upload your prompts** (JSONL format) via drag-and-drop
5. **Define reward function** or upload existing Python file
6. **Configure settings** (completion length, epochs, learning rate)
7. **Launch training** and monitor real-time metrics
8. **Track costs** and cancel anytime via the dashboard
